# PastLead

**PastLead** は、メール履歴をローカルで分析・可視化し、人脈資産を再発見するためのツールです。
Google Takeoutなどでエクスポートしたメールアーカイブ（Mbox形式）を読み込み、連絡先との関係性をスコアリングしたり、スレッド形式で閲覧したりできます。

すべてのデータはローカルPC内（Dockerコンテナ内）で処理・保存されるため、外部にプライバシー情報が流出することはありません。

## クイックスタート（一発インストール）

[Docker Desktop](https://www.docker.com/products/docker-desktop) がインストールされていれば、以下の手順で簡単にアプリを起動できます。

1.  ターミナルを開きます。
2.  メールデータ（`.mbox` ファイル）が置いてあるフォルダに移動します。
3.  以下のコマンドをコピー＆ペーストして実行してください：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/egawahideaki/pastlead/main/install.sh)"
```

> **注意**: インストール中、macOSのセキュリティ機能により、「ターミナルがファイルにアクセスしようとしています」や「Dockerを制御しようとしています」といった許可を求めるポップアップが表示される場合があります。すべて **「許可」** をクリックして進めてください。

セットアップが完了すると、自動的にブラウザが開くか、以下のURLが表示されます：
[http://localhost:3000](http://localhost:3000)

## 主な機能

*   **連絡先スコア**: バラバラになったメールを「人」単位で集約し、やり取りの頻度や直近のアクティビティから重要度を自動算出します。
*   **スレッドビュー**: 引用返信などで途切れてしまった会話も、ヘッダー情報からAI的に再構築して表示します。
*   **非表示リスト**: 不要なメルマガや通知メールのドメインを簡単に非表示設定できます。
*   **ローカル完結**: すべての解析はあなたのPC上で行われます。APIキーの設定は不要、追加料金もかかりません。

## 動作要件

*   Docker Desktop (起動していること)
*   `.mbox` ファイル (Google Takeout等で取得したもの)
    *   ファイルのディスク容量以上の空き容量が必要です（解析のために一時ファイルは作成しませんが、Docker用の領域を使用します）。
